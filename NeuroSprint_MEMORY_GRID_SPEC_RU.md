# NeuroSprint — Спека для игры Memory Grid Rush
Версия: 1.0

Цель: реализовать второй ключевой тренажёр — развитие рабочей памяти (оперативной), концентрации и скорости воспроизведения.

Статус: Вторая “идеальная” игра после Decision Rush.
Фокус: рабочая память + устойчивость + прогрессируемая сложность.

---

1. КОНЦЕПЦИЯ

Memory Grid Rush — тренажёр, в котором игрок запоминает последовательность подсвеченных клеток на сетке и воспроизводит её.

Игра должна:
- быть понятной ребёнку
- быть сложной для взрослого
- иметь режим на время
- иметь соревновательный потенциал
- масштабироваться по сложности

---

2. UX-ПОТОК

Главная → Memory Grid Rush → Setup → Session → Result → (Ещё / Лидерборд / Статистика)

Setup:
- Режим: Classic / Rush 60 sec
- Сетка: 3×3 (по умолчанию) / 4×4 (Pro)
- Уровень: Kids / Standard / Pro
- Кнопка Начать
- Кнопка Как играть

---

3. МЕХАНИКА

Каждый уровень:
1) Показ последовательности (500–800мс подсветка)
2) Короткая пауза (400мс)
3) Игрок воспроизводит кликами
4) Проверка

Classic:
- Ошибка = конец игры
- Длина +1 после успеха

Rush:
- Таймер 60 сек
- Ошибка = -1 уровень (не ниже стартового)

---

4. СЛОЖНОСТЬ

Kids:
- 3×3
- старт 3
- медленный показ

Standard:
- 3×3 → 4×4 после span ≥ 6
- быстрее показ
- таймер на ввод

Pro:
- 4×4
- старт 4
- быстрый показ
- позже добавить обратный порядок

---

5. МЕТРИКИ

Сохраняем:
- spanMax
- levelsCompleted
- errors
- avgRecallTimeMs
- accuracy
- score

Classic score:
score = spanMax*100 + levelsCompleted*20 - errors*15

Rush score:
score = levelsCompleted*50 + spanMax*40 - errors*10

---

6. ЛИДЕРБОРД

Classic:
- сортировка по spanMax (DESC)
- при равенстве — avgRecallTimeMs (ASC)

Rush:
- сортировка по score (DESC)

Best-of-period правило.

---

7. СТАТИСТИКА

- График spanMax (7-дневное сглаживание)
