
# NeuroSprint
## Когнитивный тренажёр скорости мышления
Версия документа: 1.0  
Статус: Проектирование MVP

---

# 1. Общая идея проекта

NeuroSprint — это цифровой когнитивный тренажёр для детей (первый фокус — 3 класс), 
направленный на развитие:

- скорости обработки информации
- скорости реакции
- концентрации внимания
- рабочей памяти
- скорости решения математических задач
- когнитивной гибкости

Проект создаётся как:

1. Личный тренажёр для домашнего использования
2. Инструмент для использования в школе (на ноутбуке или интерактивной доске)
3. Потенциальная масштабируемая EdTech-платформа

Основная философия:
Это не игра ради развлечения.  
Это измеряемый тренажёр когнитивной скорости.

---

# 2. Формат приложения

Приложение реализуется как:

PWA (Progressive Web App)

Это означает:

- Работает в браузере
- Устанавливается на Android как обычное приложение
- Работает офлайн
- Не требует сервера на этапе MVP
- Может быть размещено онлайн для школы

# 4. Первый модуль: Таблица Шульте

Это базовый модуль NeuroSprint.

Реализуются 2 режима:

------------------------------------
## 4.1 Classic Mode (контрольный режим)

Описание:

- Сетка 5×5
- Числа 1–25
- Секундомер
- Нужно найти числа по порядку от 1 до 25

Цель:
Измерение чистой скорости зрительного поиска.

Метрики:

- durationMs (время прохождения)
- errors (ошибки)
- speed = 25 / minutes
- accuracy
- score

Формула оценки:

score = speed × (0.7 + 0.3 × accuracy)

Этот режим используется как контрольный тест.
По нему строятся основные графики прогресса.

------------------------------------
## 4.2 Timed Mode (режим тренировки)

Описание:

- Сетка 5×5
- Таймер: 30 / 60 / 90 секунд
- После правильного нажатия:
  - в этой клетке появляется следующее число
  - nextSpawn увеличивается
- Игрок может пройти больше 25 чисел

Цель:
Тренировка темпа обработки информации.

Метрики:

- correctCount
- errors
- effectiveCorrect = correctCount - errors × 0.5
- speed
- accuracy
- score

Ошибки не блокируют управление.
Штраф влияет только на итоговый результат.

# 6. Статистика

Экран статистики должен:

Classic:
- Лучшее время по дням
- Среднее время по дням

Timed:
- effectiveCorrect в минуту
- score по дням

Группировка по локальной дате (YYYY-MM-DD).

# 7. UX требования

Интерфейс должен быть:

# 9. Философия NeuroSprint

Это тренажёр темпа мышления.

Принципы:

- Скорость важна, но не в ущерб точности
- Ошибки должны влиять на результат
- Прогресс должен быть измеряемым
- Classic — измерение
- Timed — тренировка
