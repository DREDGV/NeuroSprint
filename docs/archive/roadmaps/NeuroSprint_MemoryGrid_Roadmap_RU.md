## 📌 Заметки для агента

### Важные принципы
1. **Не ломать существующее**: Перед изменениями запускать `npm test`
2. **Тесты обязательны**: Любая новая функция = новые тесты
3. **Документация**: Обновлять этот документ после каждого этапа
4. **Коммиты**: Короткие, понятные сообщения с префиксом (feat/fix/docs)

### Контекст проекта
- **Фреймворк**: React 18 + TypeScript
- **Сборка**: Vite
- **Тесты**: Vitest + Playwright
- **Хранение**: IndexedDB (Dexie)
- **Стиль**: CSS (без препроцессоров)

- Основной агент: работает над v0.7.C (Decision Rush)
- Второй агент: работает над Memory Grid Rush
