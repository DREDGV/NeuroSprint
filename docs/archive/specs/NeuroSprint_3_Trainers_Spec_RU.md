
# NeuroSprint — Техническая спецификация тренажёров (v1.0)

Документ предназначен для ИИ‑агента разработки.
Реализовать три когнитивных тренажёра в существующем приложении NeuroSprint.

Тренажёры:
1. Memory Match
2. Spatial Memory
3. Block Pattern Recall

Работать по этапам. После каждого этапа — коммит и краткое описание выполненной работы.

--------------------------------------------------
# Общие требования для всех тренажёров

Каждый тренажёр должен:

• запускаться из меню тренировок
• иметь экран старта
• иметь экран игры
• иметь экран результата
• сохранять результат сессии
• отображаться в статистике

Структура результата:

{
 trainerId,
 userId,
 duration,
 score,
 accuracy,
 errors,
 timestamp
}

UI элементы:

- название тренажёра
- игровое поле
- таймер
- индикатор прогресса
- кнопка старт
- экран результатов

# Тренажёр 1 — Memory Match

Тип навыка:
Визуальная память

## Механика

На поле расположены карточки рубашкой вверх.

Игрок открывает две карточки.

Если карточки совпали:
они остаются открытыми.

Если не совпали:
они переворачиваются обратно через 1.5 секунды.

Игра заканчивается когда найдены все пары.

## Уровни сложности

easy
4x4 поле

medium
6x6 поле

hard
8x8 поле

## Дополнительные режимы

Classic
стандартная игра

Speed
карточки показываются 2 секунды затем закрываются

Shuffle
после найденной пары карточки перемешиваются

## Метрики

moves
time
accuracy
memoryEfficiency

memoryEfficiency = найденные пары / открытые карточки

## Definition of Done

✔ карточки переворачиваются
✔ совпадение работает
✔ карточки исчезают / фиксируются
✔ игра завершается
✔ результат сохраняется

# Тренажёр 2 — Spatial Memory

Тип навыка:
Пространственная память

## Механика

Фаза 1 — Показ сцены

На поле появляются объекты.

пример:
дерево
камень
книга
куб

Фаза 2 — Скрытие

объекты исчезают.

Фаза 3 — Воспроизведение

игрок должен восстановить их расположение.

## Уровни сложности

easy
3 объекта

medium
5 объектов

hard
8 объектов

## Режим Rotation

карта поворачивается:

90°
180°
270°

игрок должен указать положение объектов после поворота.

## Режим Movement

объекты двигаются
игрок должен запомнить конечную позицию.

## Метрики

distanceError
accuracy
time
objectsCorrect

distanceError вычисляется как дистанция между правильной и выбранной позицией.

## Definition of Done

✔ сцена отображается
✔ объекты исчезают
✔ пользователь может разместить объекты
✔ система считает ошибку
✔ результат сохраняется

# Тренажёр 3 — Block Pattern Recall

Тип навыка:
Рабочая память + пространственное мышление

## Игровое поле

4x4 сетка блоков

## Механика

Фаза 1 — Показ

На поле появляется фигура из блоков.

пример:

■ □ □ □
■ ■ □ □
□ ■ □ □
□ □ □ □

Игрок должен запомнить фигуру.

Фаза 2 — Скрытие

Фигура исчезает.

Фаза 3 — Воспроизведение

Игрок отмечает блоки которые были подсвечены.

## Уровни сложности

easy
3 блока

medium
4–5 блоков

hard
6–7 блоков

## Дополнительные режимы

Rotation

поле поворачивается:

90°
180°
270°

игрок должен выбрать блоки с учётом поворота.

Mirror

фигура отображается зеркально.

Speed

время показа уменьшается

3 сек
2 сек
1 сек

## Метрики

patternSize
accuracy
reactionTime
rotationCorrect

## Definition of Done

✔ генерация случайных фигур
✔ отображение фигуры
✔ скрытие фигуры
✔ ввод игрока
✔ проверка результата
✔ сохранение результата
