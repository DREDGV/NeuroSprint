
# NeuroSprint — UX спецификация тренажёров
Версия: 1.0

Документ предназначен для ИИ‑агента разработки.
Описывает внешний вид, поведение интерфейса и логику экранов
для трех тренажеров:

1. Memory Match
2. Spatial Memory
3. Block Pattern Recall

Цель: единый современный интерфейс, подходящий для детей и взрослых,
без перегруженной или "казино" стилистики.

--------------------------------------------------
# Общий стиль приложения

Стиль:
минимализм + образовательная среда

Основные принципы:

• спокойные цвета
• четкие формы
• крупные элементы
• отсутствие визуального шума

Цветовая палитра:

Primary: #2F6BFF
Secondary: #00A9A5
Accent: #F7B500
Background: #F4F6FA
Card: #FFFFFF
Error: #FF4D4F
Success: #52C41A

Шрифт:

Primary font: Inter / Roboto
Крупный жирный для цифр и блоков.

--------------------------------------------------
# Структура экрана тренажёра

Каждый тренажёр имеет одинаковую структуру.

Экран 1 — Start Screen

Элементы:

Название тренажёра
Краткое описание
Кнопка Start
Выбор сложности

Экран 2 — Game Screen

Элементы:

Header:
таймер
прогресс

Main:
игровое поле

Footer:
подсказка или сообщение

Экран 3 — Result Screen

Показывать:

Score
Accuracy
Time
Кнопка повторить
Кнопка назад

--------------------------------------------------
# Memory Match UX

Поле карточек строго квадратное.

Размер карточки:

Mobile:
70–90px

Desktop:
90–120px

Карточка:

закрытая — светлая
открытая — белая
совпадение — зеленая подсветка
ошибка — красная подсветка

Анимации:

flip animation
200ms

закрытие карточек
1500ms

--------------------------------------------------
# Spatial Memory UX

Игровое поле — квадратная карта.

Размер поля:

Mobile:
300px

Desktop:
420px

Объекты:

простые иконки
дерево
куб
камень
книга

Фаза запоминания:

объекты подсвечены
таймер обратного отсчета

Фаза воспроизведения:

пустое поле
игрок размещает объекты

--------------------------------------------------
# Block Pattern Recall UX

Сетка:

4x4 блоки

Размер блока:

Mobile:
70px

Desktop:
90px

Фаза показа:

блоки подсвечены синим

Фаза ввода:

пустая сетка

Выбранный блок:

синий

Правильный блок:

зеленый

Ошибка:

красный

--------------------------------------------------
# Анимации

Все анимации мягкие.

Fade in:
200ms

Fade out:
200ms

Highlight:
150ms

Rotation animation:
250ms

--------------------------------------------------
# Поведение ошибок

Если пользователь ошибся:

блок подсвечивается красным
краткая вибрация (на телефоне)
показывается правильный ответ

--------------------------------------------------
# Поведение успеха

Если задача решена:

зеленая подсветка
короткая анимация успеха
звук успеха

--------------------------------------------------
# Требования к адаптивности

Интерфейс должен работать:

телефон
планшет
ноутбук
проектор в классе

Поле всегда квадратное.
