# NeuroSprint Alpha v0.3.0 (PWA)

Дата фиксации: 2026-02-24

## Что вошло в релиз
- Зафиксирован этап `v0.3.A` по расширенной статистике и сравнительному анализу.
- В `Настройки` добавлен генератор демо-класса:
  - 2 группы,
  - 30 учеников,
  - 14 дней сессий,
  - быстрый старт для проверки аналитики на больших локальных данных.
- Расширены сценарии сравнения:
  - Индивидуальная статистика: `Я / другой пользователь / моя группа / все пользователи`.
  - Групповая статистика: `текущая группа / группа для сравнения / все пользователи`.
- Улучшена групповая аналитика:
  - лучший/средний/худший,
  - перцентиль ученика,
  - распределение уровней (гистограмма),
  - фильтры периода.
- Старт `v0.3.B` (hardening/performance):
  - оптимизированы выборки по compound index `[userId+moduleId+modeId]`,
  - фильтрация периода переведена на `localDate` (меньше лишних преобразований дат).

## Тесты и качество
- `npm test` — успешно.
- `npm run build` — успешно.
- `npm run test:e2e` — успешно.
- Добавлены новые проверки:
  - integration для блоков сравнений (`stats individual/group`),
  - e2e сценарий генерации fixture и проверки сравнений.

## Ограничения alpha
- Данные по-прежнему полностью локальные (без backend/облака).
- Групповая аналитика рассчитана на локальную работу класса на одном устройстве.
