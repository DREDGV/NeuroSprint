# Android: как тестировать сейчас и где APK

## Коротко
Сейчас проект реализован как **PWA**.  
Поэтому отдельного `APK` в репозитории пока нет.

## Как протестировать на Android (без APK)
1. Запустите локально на ПК:
```powershell
npm run build
npm run preview -- --host 0.0.0.0 --port 4173
```
2. Откройте на телефоне тот же URL в вашей сети:
`http://<IP_ПК>:4173`
3. В Chrome на Android выберите:
`Меню -> Установить приложение`.
4. После установки:
   - откройте приложение из иконки;
   - закройте сеть на телефоне;
   - убедитесь, что приложение открывается офлайн.

## Если нужен именно APK
Есть 2 рабочих пути:
1. `TWA (bubblewrap)`:
   - быстрее для уже готовой PWA;
   - APK открывает ваш web-app.
2. `Capacitor`:
   - web-код упаковывается в Android-проект;
   - больше контроля, но больше инфраструктуры.

## Рекомендация для текущего этапа
Сначала закрыть Android smoke-check через PWA.  
После этого решить, нужен ли APK для пилота, и выбрать TWA как более быстрый старт.
