# NeuroSprint Changelog (RU)

Формат: фиксируем версии, даты и ключевые изменения, чтобы быстро восстановить контекст по релизам и dev-этапам.

## [0.7.0-dev.6] - 2026-04-12
### Добавлено и изменено
- Добавлены глобальная форма `Отзыв`, `post-session feedback`, публичная доска идей и раздел `Мои идеи`.
- Введён базовый moderation workflow для идей: pending, approved, rejected, roadmap-статусы и отдельная страница `/admin/ideas`.
- Появился foundation для `site role` (`user | moderator | admin`) и новая миграция ролей/feature flags.
- Улучшен UX доски идей: одна кнопка отмены, success-state после отправки, понятная поддержка идеи, запрет голосования за собственную идею.
- `FeedbackModal` переработан визуально: новые категории, более аккуратный layout, понятные локальные ограничения на localhost.
- Для локальной разработки связанные профили на `localhost` больше не блокируют тестирование вкладок без облачного входа.
- Настройки получили новый секционный интерфейс: общие настройки, аудио, профиль, экспорт и devtools.
- Дополнительно русифицированы названия тренажёров и режимов в каталоге, pre-session, статистике, daily challenge, skill recommendations и тестах.

### Auth и recovery
- Стабилизирован recovery-flow: ссылка из письма приводит к экрану смены пароля, а не к повторному запросу email.
- Логика сброса пароля и recovery redirect доведена до рабочего состояния на боевом домене.

### Внутренняя инфраструктура
- Усилен runtime для `api/feedback`, `api/ideas` и admin-ендпоинтов.
- Добавлены миграции:
  - `002_feedback_and_ideas.sql`
  - `003_roles_and_permissions.sql`

## [0.7.0-dev.5] - 2026-04-10
### Изменено
