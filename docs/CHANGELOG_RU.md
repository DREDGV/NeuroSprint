# NeuroSprint Changelog (RU)

Формат: фиксируем версии, даты и ключевые изменения для быстрого восстановления контекста.

## [0.7.0-dev.5] - 2026-04-10
### Изменено
- Локально подтверждены основные auth-сценарии: регистрация, импорт профиля, выход и повторный вход.
- Страница восстановления доступа теперь корректно распознаёт recovery-ссылки и показывает форму нового пароля.
- Ошибки авторизации и ограничение на отправку писем переведены в понятные пользовательские сообщения.
- Supabase-миграция аккаунтов и профилей зафиксирована как отдельный source of truth для развёртывания.

### Следующий шаг
- Проверить тот же сценарий уже на реальном домене после деплоя: `register -> import -> logout -> login -> forgot password`.

## [0.7.0-dev.4] - 2026-04-09
### Изменено
- Система аккаунтов и профилей доведена до public-ready сценария вокруг единого центра `Профили и аккаунт`.
- Главная страница больше не дублирует вход и регистрацию, а ведёт в единый сценарий выбора профиля и аккаунта.
- В `/profiles` добавлены смена имени, запуск смены email и отправка письма для смены пароля.
- Supabase-схема аккаунтов и профилей приведена к одному корректному source of truth для миграций и ручной настройки.

### Документация
- Обновлены справка, changelog и версия приложения.
- В справку и на главную страницу добавлена пометка, что проект находится в активной разработке.
- Добавлены e-mail для обратной связи: `dr-edgv@yandex.ru`, `edgvaud@gmail.com`.

### Проверки
- `npm run lint` — passed
- `npm run build` — passed

## [0.7.0-dev.3] - 2026-04-04
### Изменено
- Русские названия всех тренажёров: Математический спринт, Реакция, N-Back, Сетка памяти, Пары памяти, Пространственная память, Быстрые решения, Распознавание паттернов.
- Развёрнутые описания тренажёров: каждое объясняет что тренирует модуль и как это помогает в жизни.
- Система соревнований: PvP вызовы, командные челленджи, лидерборды, уведомления.
- WebSocket клиент для real-time событий как подготовка к синхронным соревнованиям.
- Подключена production-аналитика: `Vercel Analytics` и `Speed Insights`.
- Стабилизирован `feature flag store`.
- Справка вынесена из основной навигации в отдельную кнопку `?` рядом с бейджем роли.
- Добавлены новые виджеты классов: тепловая карта активности, радар навыков, топ учеников, лидерборды.
- Исправлено: в `TrainingHub` убран дублирующий рендер и оптимизирована структура карточек навыков.
- Исправлено: `RequirePermission` получил отсутствующий import.

## [0.7.0-dev.1] - 2026-03-12
### Изменено
- `Spatial Memory` переведён из experimental-блока в основной каталог тренажёров.
