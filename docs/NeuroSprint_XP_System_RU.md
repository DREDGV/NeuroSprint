## 📊 Принципы системы

### 1. **Справедливость**
- Больше усилий = больше XP
- Сложные режимы = больше XP
- Точность важна (не просто клики)

### 2. **Прогрессия**
- Первые уровни быстрые (мотивация)
- Поздние уровни медленнее (вызов)
- Нет "потолка" (бесконечный рост)

### 3. **Баланс**
- Не наказывать за ошибки (демотивация)
- Поощрять точность и серии
- Разные тренажёры = разная сложность

## 🎯 Формула начисления XP

### Базовая формула
```
XP = BaseXP × DifficultyMultiplier × AccuracyMultiplier × ComboMultiplier
```

### Параметры

| Параметр | Описание | Значения |
|----------|----------|----------|
| **BaseXP** | Базовое XP за тренажёр | 10-50 |
| **DifficultyMultiplier** | Множитель сложности | 1.0 / 1.5 / 2.0 |
| **AccuracyMultiplier** | Множитель точности | 0.5 - 1.5 |
| **ComboMultiplier** | Множитель серии | 1.0 - 2.0 |

## 🎯 Множитель точности (Accuracy)

```
Accuracy = correctCount / totalAttempts

Accuracy    | Multiplier
------------|------------
< 50%       | 0.5× (минимум)
50-60%      | 0.7×
60-70%      | 0.85×
70-80%      | 1.0× (база)
80-90%      | 1.2×
90-95%      | 1.35×
95-100%     | 1.5× (максимум)
```

**Пример:**
- Игрок набрал 85% точности → ×1.2
- BaseXP 30 × 1.2 = **36 XP**

## 🔥 Множитель серии (Combo)

### Daily Streak (серия дней)
```
Streak Days | Multiplier
------------|------------
1 день      | 1.0×
2-3 дня     | 1.1×
4-7 дней    | 1.25×
8-14 дней   | 1.5×
15-30 дней  | 1.75×
30+ дней    | 2.0× (максимум)
```

### Session Combo (серия правильных в сессии)
```
Combo | Multiplier
------|------------
1-5   | 1.0×
6-10  | 1.1×
11-20 | 1.25×
21-50 | 1.5×
50+   | 1.75×
```

## 📊 Примеры расчётов

### Пример 1: Memory Grid Rush (Standard)
```
BaseXP: 30 XP
Difficulty: Standard (1.3×)
Accuracy: 88% (1.2×)
Daily Streak: 5 дней (1.25×)

XP = 30 × 1.3 × 1.2 × 1.25 = 58.5 XP → **58 XP**
```

### Пример 2: N-Back Lite (3-back, 4×4)
```
BaseXP: 25 XP
Difficulty: 3-back (1.8×) + 4×4 (0.2×) = 2.0×
Accuracy: 72% (1.0×)
Daily Streak: 1 день (1.0×)

XP = 25 × 2.0 × 1.0 × 1.0 = **50 XP**
```

### Пример 3: Шульте (Reverse, 4×4)
```
BaseXP: 15 XP
Difficulty: Reverse (1.5×) + 4×4 (0.3×) = 1.8×
Accuracy: 95% (1.35×)
Daily Streak: 15 дней (1.75×)

XP = 15 × 1.8 × 1.35 × 1.75 = 63.8 XP → **64 XP**
```

### Формула XP для уровня
```
XP для уровня N = 100 × N^1.5

Пример:
Уровень 5: 100 × 5^1.5 = 1,118 XP (округлено до 1,000)
Уровень 10: 100 × 10^1.5 = 3,162 XP (округлено до 12,000 накопленных)
```

## 📊 Баланс и тестирование

### Целевые показатели
| Метрика | Цель |
|---------|------|
| Время до уровня 5 | 3-5 дней |
| Время до уровня 10 | 2-3 недели |
| Время до уровня 15 | 2-3 месяца |
| Средняя сессия | 2-4 минуты |
| XP за сессию | 30-80 XP |

### Тестовые сценарии
1. **Новичок** (1 сессия/день, 5 мин)
   - День 1: ~50 XP → Уровень 1
   - День 7: ~350 XP → Уровень 2-3
   - День 30: ~1,500 XP → Уровень 4-5

2. **Активный** (3 сессии/день, 15 мин)
   - День 1: ~150 XP → Уровень 1-2
   - День 7: ~1,050 XP → Уровень 3-4
   - День 30: ~4,500 XP → Уровень 7-8

3. **Хардкор** (10 сессий/день, 50 мин)
   - День 1: ~500 XP → Уровень 3-4
   - День 7: ~3,500 XP → Уровень 6-7
   - День 30: ~15,000 XP → Уровень 10-11
