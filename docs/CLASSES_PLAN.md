# 🏫 НейроСпринт: Система Классов и Соревнований

## 📋 Описание проекта

**НейроСпринт** — это образовательная платформа с когнитивными тренажёрами для развития памяти, внимания, реакции и мышления.

**Цель развития вкладки "Классы"** — превратить её из простого списка учеников в центр соревновательной активности с поддержкой онлайн-соревнований в реальном времени.

---

## 🎯 Миссия

> Сделать обучение увлекательным через здоровую конкуренцию, визуализацию прогресса и командные соревнования.

## 🔧 Технические детали

### Модель данных (IndexedDB → PostgreSQL)

```typescript
// Классы
db.classGroups       // id, name, createdBy, memberIds, settings
db.classStats        // cached statistics

// Соревнования
db.competitions      // id, name, type, mode, participants, results
db.challenges        // PvP вызовы
db.competitionTeams  // Команды

// Рейтинг
db.seasonRatings     // Сезонный рейтинг
db.seasons           // Информация о сезонах

// Уведомления
db.notifications     // Уведомления пользователя
```

### API Endpoints (будущий backend)

```
POST   /api/competitions          — Создать соревнование
GET    /api/competitions          — Список соревнований
GET    /api/competitions/:id      — Детали соревнования
POST   /api/competitions/:id/join — Присоединиться
POST   /api/competitions/:id/submit — Отправить результат

POST   /api/challenges            — Создать вызов
GET    /api/challenges            — Список вызовов
POST   /api/challenges/:id/respond — Ответить на вызов

GET    /api/leaderboards          — Лидерборды
GET    /api/seasons               — Сезоны
GET    /api/seasons/:id/rating    — Рейтинг сезона

WS     /ws/competitions           — WebSocket для real-time
```

### Frontend Architecture

```
src/
├── entities/
│   ├── class/           — ClassGroup, ClassStats
│   ├── competition/     — Competition, Participant
│   └── challenge/       — UserChallenge
├── features/
│   └── competitions/
│       ├── components/  — UI компоненты
│       ├── hooks/       — React хуки
│       └── services/    — API calls
├── widgets/
│   ├── ClassDashboardWidget.tsx
│   ├── LeaderboardWidget.tsx
│   └── SeasonRatingWidget.tsx
└── shared/
    └── types/
        └── classes.ts   — Типы данных
```

## 📊 Метрики успеха

### Для учеников:
| Метрика | Цель |
|---------|------|
| % активных за неделю | >60% |
| Среднее число сессий | >3 в неделю |
| Участие в соревнованиях | >40% |
| Удовлетворённость | >4/5 |

### Для учителей:
| Метрика | Цель |
|---------|------|
| Время на создание соревнования | <2 мин |
| Просмотр статистики класса | 1 раз в неделю |
| Проведение соревнований | 1-2 в месяц |

## 🚀 Риски и решения

| Риск | Решение |
|------|---------|
| Сложность backend | Начать с async соревнований |
| Нагрузка на сервер | Redis кэш + rate limiting |
| Читерство | Валидация на сервере + аномалии |
| Низкая активность | Геймификация + уведомления |
