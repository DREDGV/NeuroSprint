# NeuroSprint Alpha v0.2.0 (PWA)

Дата фиксации: 2026-02-24

## Что вошло в релиз
- Новая навигация и IA:
  - `Главная | Тренировки | Статистика | Профили | Настройки`.
  - Отдельный раздел тренировок: `/training`, `/training/schulte`, `/training/schulte/:mode`.
- Расширенный модуль Шульте:
  - `Classic+` с выбором сетки `4x4 | 5x5 | 6x6`.
  - `Timed+` с лимитами `30 | 45 | 60 | 90`, штрафом и стратегией спавна.
  - Новый режим `Reverse`.
- Единый UX запуска:
  - Настройка режима через пресеты/расширенные параметры.
  - Старт сессии только по кнопке `Начать`.
- Адаптивная система сложности:
  - Профиль уровня по связке `пользователь + модуль + режим`.
  - Пересчет каждые 3 сессии.
  - Поддержка авто- и ручного режима (`auto|manual`).
- Улучшенная персональная статистика:
  - Динамика по режимам.
  - Инсайты по серии дней.
  - Сравнение недели к неделе.
  - Рекомендация режима на сегодня.
- Подготовка к групповой аналитике:
  - Расширение схемы Dexie до `version(3)`.
  - Добавлены таблицы `classGroups` и `groupMembers`.

## Обратная совместимость
- Старые маршруты поддерживаются редиректами:
  - `/play/schulte/classic` -> `/training/schulte/classic_plus`
  - `/play/schulte/timed` -> `/training/schulte/timed_plus`
- Legacy-сессии при миграции получают дефолты:
  - `moduleId=schulte`, `modeId=classic|timed`, `level=1`, `presetId=legacy`.
