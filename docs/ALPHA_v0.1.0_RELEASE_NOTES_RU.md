# NeuroSprint Alpha v0.1.0 (PWA)

Дата фиксации: 2026-02-23

## Что включено
- Профили пользователей (создание, переименование, удаление).
- Активный пользователь в `localStorage` (`ns.activeUserId`).
- Таблица Шульте:
  - `Classic` (1..25, таймер, ошибки, score).
  - `Timed` (30/60/90 сек, nextSpawn, штраф ошибок, score).
- Локальное хранение сессий в IndexedDB (`Dexie`).
- Экран статистики с агрегацией по дням.
- Настройки Timed (лимит по умолчанию и штраф).
- PWA-конфигурация (`manifest`, service worker).

## Проверка качества (на момент фиксации)
- `npm run build` — успешно.
- `npm test` — успешно.
- `npm run test:e2e` — успешно (через системный `msedge`).

## Ограничения текущей альфы
- Это PWA, не нативный APK.
- Android-проверка установки/офлайна должна быть выполнена вручную.
- Нет серверной части и синхронизации.
