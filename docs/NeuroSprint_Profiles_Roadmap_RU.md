## 📌 Заметки для агента

### Важные принципы
1. **Не дублировать статистику**: StatsPage уже есть — только мини-сводка
2. **Сохранять доступность**: Ролевая модель должна работать
3. **Тесты обязательны**: Любое изменение = проверка тестов
4. **Контекст для других агентов**: Не ломать Decision Rush / Pattern Recognition

### Контекст проекта
- **Фреймворк**: React 18 + TypeScript
- **Сборка**: Vite
- **Тесты**: Vitest + Playwright
- **Хранение**: IndexedDB (Dexie) + localStorage
- **Стиль**: CSS (без препроцессоров)

### Зависимости
- **Memory Grid Rush**: v0.7.D готово (режимы сложности)
- **Decision Rush**: другой ИИ работает
- **Pattern Recognition**: другой ИИ работает

### Синхронизация
- Перед слиянием: `npm run check:encoding`, `npm test`, `npm run build`
- Не трогать version/changelog/roadmap общие файлы
- Слияние после завершения этапа v0.7.E
